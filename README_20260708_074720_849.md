# AudioExtract — 音频提取器

> 从视频文件中快速提取音频轨道，支持多种输出格式和比特率，适用于剪辑素材音频分离场景。

## 功能列表

- **单个/批量提取**：支持一次性添加多个视频文件，逐个自动提取音频
- **多格式输出**：MP3、WAV、AAC、FLAC 四种音频格式任选
- **比特率可调**：MP3 / AAC 支持 128k / 192k / 256k / 320k 四档比特率
- **自动重名处理**：输出目录存在同名文件时自动添加序号后缀，不会覆盖
- **进度反馈**：进度条实时显示当前提取进度和完成数量
- **ffmpeg 自动检测**：启动时自动在 PATH 和常见安装路径中查找 ffmpeg，检测不到时给出明确提示

## 安装方法

### 环境要求

- Python 3.8+
- ffmpeg（需自行安装并添加到系统 PATH）

### 安装 ffmpeg（Windows）

1. 从 [ffmpeg 官网](https://ffmpeg.org/download.html) 或 [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) 下载 `ffmpeg-release-full.7z`
2. 解压到 `C:\ffmpeg\`，确保 `C:\ffmpeg\bin\ffmpeg.exe` 存在
3. 将 `C:\ffmpeg\bin` 添加到系统 PATH 环境变量，或直接将 ffmpeg.exe 放在项目目录中

### 获取项目

```bash
git clone git@github.com:solcat1007/audio-extract.git
cd audio-extract
```

无需安装额外 Python 依赖，仅使用 Python 标准库。

## 详细使用

### 启动

```bash
python audio_extract.py
```

### 操作步骤

1. 点击 **「添加视频」** 选择一个或多个视频文件（支持 MP4 / MOV / AVI / MKV / WMV / FLV / WebM / M4V / TS / MTS / M2TS）
2. 在文件列表中选中不需要的文件，点击 **「移除选中」** 或 **「清空列表」**
3. 在输出设置中选择 **输出格式**（MP3 / WAV / AAC / FLAC）和 **比特率**（MP3/AAC 适用；WAV/FLAC 无此选项）
4. 点击 **「开始提取音频」** 开始处理
5. 音频文件输出到源视频所在目录，与视频同名但扩展名不同

### 使用示例

**场景 1：提取单个视频的背景音乐**

```
1. 添加视频 D:\素材\航拍.mp4
2. 格式选择 mp3，比特率 320k
3. 点击开始提取 → 输出 D:\素材\航拍.mp3
```

**场景 2：批量提取多个视频的音频**

```
1. 一次性选中 10 个视频文件
2. 格式选择 wav（无损），比特率无需设置
3. 点击开始提取 → 输出 10 个同目录 .wav 文件
4. 进度条逐项推进，完成后弹窗汇总成功/失败数量
```

## 输入输出说明

### 支持的输入格式

| 容器格式 | 扩展名 |
|---|---|
| MP4 | `.mp4` |
| QuickTime | `.mov` |
| AVI | `.avi` |
| Matroska | `.mkv` |
| Windows Media | `.wmv` |
| Flash Video | `.flv` |
| WebM | `.webm` |
| iTunes Video | `.m4v` |
| MPEG Transport Stream | `.ts` / `.mts` / `.m2ts` |

### 输出格式

| 格式 | 编码器 | 比特率可选 | 备注 |
|---|---|---|---|
| MP3 | libmp3lame | 128k / 192k / 256k / 320k | 通用兼容性最佳 |
| WAV | PCM 16-bit | 不适用 | 无损，文件较大 |
| AAC | aac | 128k / 192k / 256k / 320k | 现代格式，体积小 |
| FLAC | flac | 不适用 | 无损压缩，比 WAV 小 |

### 输出路径

音频文件输出到**源视频所在目录**，文件名规则：
- 默认：`视频文件名.输出格式扩展名`
- 若已有同名文件：`视频文件名_1.输出格式扩展名`、`视频文件名_2.输出格式扩展名` …

## 注意事项

1. ffmpeg 必须已安装且可被检测到（PATH 或 `C:\ffmpeg\bin\ffmpeg.exe`、`C:\Program Files\ffmpeg\bin\ffmpeg.exe`）
2. WAV 和 FLAC 为无损格式，提取时自动使用源音频的最佳质量，无需也无"比特率"概念
3. 当前版本默认提取第一个音频轨道。如需选择特定音轨，请使用 ffmpeg 命令行手动处理（`-map 0:a:1` 等）
4. 提取的音频质量取决于源视频本身的音频质量
